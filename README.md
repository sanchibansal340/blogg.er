# Blogg.er
Blogging platform so that you can create, read and delete blogs easily

## Technology used
- React JS
- Bootstrap
- MongoDB
- Node.js 
- Express JS
- Passport JS (JWT strategy)

## Funtions
- Create a user profile
- Create, Read, Update and Delete a blog
